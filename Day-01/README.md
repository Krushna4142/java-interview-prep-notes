Developer writes .java file

Java Compiler converts it into bytecode

JVM loads the bytecode

JVM executes it on the current OS

<h3>ğŸ¯ Interview Preparation (Day 01 Level)</h3>

Q1. What is Java?
Java is an object-oriented, platform-independent programming language used to build scalable applications.

Q2. What is JVM?
JVM is a virtual machine that executes Java bytecode and enables platform independence.

Q3. Difference between JVM, JRE, and JDK?
JVM runs the program, JRE provides runtime environment, and JDK is used for development.

Q4. Why is Java platform independent?
Because Java code is compiled into bytecode which can run on any system having a JVM.
